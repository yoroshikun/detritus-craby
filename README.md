## Detritus Craby

Discord bot for the Citizens of WKClass build with nodejs + [detritus][detritus_web]

## Requirements

- Nodejs runtime (>=16.0.0, for 'fs/promises')

## Development

```bash
npm install
npm run dev
```

## Deployment

```bash
npm compile
npm start
```

[detritus_web]: https://detritusjs.com/
